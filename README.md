# Proteome-scale prediction of phosphorytaion events that disrupt protein-protein interactions
Systematically mapped thousands of phosphorylation events from budding yeast proteome on to 3D structures and identified modifications that could negatively regulate protein-protein interactions.

**Link to the published peer-reviewed article in EMBO Reports journal here: https://link.springer.com/article/10.15252/embr.202051121**

**Summary**
This study compiles a large set of independent experiments into a comprehensive phosphoproteome resource for the budding yeast community. 3D analysis of protein interaction interfaces and other strategies are used to predict functionality amongst the â‰¥ 40,000 reported phosphorylation events.

* 75 independent phosphoproteomic experiments were consolidated into a comprehensive resource of over 40,000 budding yeast phosphorylation sites.

* Multiple strategies were used to infer functional phosphorylation events.

* Mapping phosphorylation sites to protein interaction interfaces revealed phosphorylation sites that regulate protein-protein interactions.
